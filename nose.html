Instalación de CrotoSafe

Lo primero que debes tener en cuenta, esque este paquete no cuenta con un alojamiento 24/7 por lo tanto si se cierra se desactivará, pero uno de los próximos objetivos para la actualización es ese.

Paso 1

Instala CrotoSafe globalmente

npm install -g crotosafe

Paso 2

Inicia CrotoSafe

crotosafe

Y se iniciará el CrotoSafe.

Comandos Disponibles

A continuación te daré los comandos que están disponibles actualmente.

  • HELP; >> Mira la lista de comandos
  • SHOW crotosafe; >> Mira las estadísticas
  • ENABLE crotosafe; >> Activa CrotoSafe

Estos son los comandos para CrotoSafe posiblemente haigan más en el menu de ayuda desde tu terminal.
